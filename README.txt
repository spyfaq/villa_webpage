Olga’s Luxury Villa website template with lightbox gallery

What changed:
- Added a full-screen lightbox viewer
- Any image with class="gallery-photo" opens in the lightbox

How to add more photos:
1. Create an images folder
2. Put your extra villa photos there
3. Replace any placeholder with:
   <img src="images/pool.jpg" class="gallery-photo" alt="Private pool area" />

Links to customize:
- WhatsApp number
- Google Maps link and embed
- Airbnb URL
- Booking.com URL
- Email and phone
